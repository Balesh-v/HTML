What is CSS?

Cascading Style Sheets (CSS) is used to format the layout of a webpage. With CSS, you can control the color, font, the size of text, the spacing between elements, how elements are positioned and laid out, what background images or background colors are to be used, different displays for different devices and screen sizes, and much more!

Using CSS

CSS can be added to HTML documents in 3 ways:

  • Inline - by using the style attribute inside HTML elements
  • Internal - by using a style element in the head section
  • External - by using a link element to link to an external CSS file

The most common way to add CSS, is to keep the styles in external CSS files. However, in this tutorial we will use inline and internal styles, because this is easier to demonstrate, and easier for you to try it yourself.

Inline CSS

An inline CSS is used to apply a unique style to a single HTML element. To use inline CSS, add the style attribute to the relevant element. The style attribute can contain any CSS property.

The following example changes the color of a paragraph to red, and adds a blue background color:

<p style="color:red;background-color:blue;">This is a paragraph.</p>

The result of the above code will be a paragraph with red text on a blue background.

Internal CSS

An internal CSS is used to define a style for a single HTML page. To use internal CSS, add a style element in the head section of the page.

The following example changes the text color of all paragraphs to red:

<style> p { color: red; } </style>

The result of the above code will be that all paragraphs on the page will have red text.

External CSS

An external CSS is used to define a style for multiple HTML pages. To use external CSS, add a link element in the head section of the page.

The following example links to an external CSS file called style.css:

<link rel="stylesheet" href="style.css">

The result of the above code will be that all styles defined in the style.css file will be applied to the HTML page.

CSS Syntax

A CSS rule-set consists of a selector and a declaration block. The selector points to the HTML element you want to style. The declaration block contains one or more declarations separated by semicolons.

Each declaration includes a CSS property name and a value, separated by a colon. The property name is the style you want to change, and the value is the new value for that style.

The following example changes the text color of all paragraphs to red and sets the font size to 20px:

p { color: red; font-size: 20px; }

The result of the above code will be that all paragraphs on the page will have red text and a font size of 20 pixels.
