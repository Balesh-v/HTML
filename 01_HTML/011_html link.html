HTML Links - Hyperlinks

HTML links are hyperlinks. You can click on a link and jump to another document. When you move the mouse over a link, the mouse arrow will turn into a little hand.

HTML Links - Syntax

The HTML <a> tag defines a hyperlink. It has the following syntax: <a href="url">link text</a>

Example

The following example creates a link to W3Schools: <a href="https://www.google.com/">Visit google</a>

Output

Visit google

HTML Links - The target Attribute

By default, the linked page will be displayed in the current browser window. To change this, you must specify another target for the link.

The target attribute specifies where to open the linked document.

The target attribute can have one of the following values:

  • _blank - Opens the linked document in a new window or tab.
  • _self - Opens the linked document in the same frame as it was clicked (this is default).
  • _parent - Opens the linked document in the parent frame.
  • _top - Opens the linked document in the full body of the window.

Example:

<a href="https://www.google.com/" target="_blank">Visit google</a>

Output

Visit google

HTML Links - Use an Image as a Link

You can use an image as a link by placing the image inside the <a> tag.

Example:

<a href="https://www.google.com/"><img src="https://www.w3schools.com/images/w3schools_green.jpg" alt="W3Schools.com" style="width: 100px; height: 100px;" /></a>

Output

Link to an Email Address

You can create a link that opens the user's email program to send an email by using the mailto: protocol.

Example:

<a href="mailto:sample@gmail.com">Send Email</a>

Output

Send Email

Link to a Phone Number

You can create a link that opens the user's phone dialer to call a number by using the tel: protocol.

Example:

<a href="tel:+1234567890">Call Us </a>

Output

Call Us

Button as a Link

You can also use a button as a link by wrapping the button inside the <a> tag.

Example:

<a href="https://www.google.com/">< button type="button">Visit Google</button></a>

Output

Link Titles

You can add a title to a link using the title attribute. The title will be displayed as a tooltip when the user hovers over the link.

Example:

<a href="01_Introduction.html" title="Visit Google">go to introduction html</a>

Output

go to introduction html
