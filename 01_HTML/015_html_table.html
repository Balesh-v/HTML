Html table

HTML tables allow web developers to arrange data into rows and columns.

Define an HTML Table

Define an Html Table in Table tag

<table> </table>

Table Rows

Each table row is defined by a <tr> and a </tr> tag.

Everything between <tr> and </tr> is the content of a table row.

<table>
    <tr>
        
    </tr>
</table>

Table Headers

Table headers are defined with the <th> tag.

Table headers are bold and centered by default.

Table headers are usually used to specify the column heading.

<table>
    <tr>
        <th> Company </th>
    </tr>
</table>

Output

Company

Table Cells

Each table cell is defined by a <td> and a </td> tag.

td stands for table data.

Everything between <td> and </td> is the content of a table cell.

<table>
    <tr>
        <th> Company </th>
    </tr>
    <tr>
        <td> selltome </td>
    </tr>
</table>

output

Company
selltome

Caption

The <caption> tag is used to add a caption to a table.

The <caption> tag must be inserted immediately after the <table> tag.

By default, the caption is centered above the table.

<table>
    <caption> Company Data </caption>
    <tr>
        <th> Company </th>
    </tr>
    <tr>
        <td> selltome </td>
    </tr>
</table>

output

Company Data
Company
selltome

Example

Here is an example of a complete HTML table with a caption, header, and multiple rows and columns:

<table>
    <caption> Company Data </caption>
    <tr>
        <th> Company </th>
        <th> Content</th>
        <th> Country</th>
    </tr>
    <tr>
        <td> selltome </td>
        <td> This is a sample content. </td>
        <td> USA </td>
    <tr>
    <tr>
        <td> buyme </td>
        <td> This is another sample content. </td>
        <td> UK </td>
    <tr>
    <tr>
        <td> rentit </td>
        <td> This is yet another sample content.</td>
        <td> India </td>
    <tr>
    <tr>
        <td> leaseit </td>
        <td> This is yet another sample content.</td>
        <td> Canada </td>
    <tr>
