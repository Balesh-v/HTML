Html List

HTML lists are used to group a set of related items in no particular order.

Types of HTML Lists

There are three types of HTML lists:

  • Ordered List - A list in which the order of the items is important. The list items are marked with numbers (1, 2, 3, etc.) or letters (A, B, C, etc.).
  • Unordered List - A list in which the order of the items is not important. The list items are marked with bullets (â€¢).
  • Description List - A list of terms and their descriptions. The terms are marked with the <dt> tag and the descriptions with the <dd> tag.

Ordered List

An ordered list is defined with the <ol> tag. Each list item is defined with the <li> tag.

<ol>
    <li> Item 1 </li>
    <li> Item 2 </li>
    <li> Item 3 </li>
</ol>

Output

  1. Item 1
  2. Item 2
  3. Item 3

Unordered List

An unordered list is defined with the <ul> tag. Each list item is defined with the <li> tag.

<ul>
    <li > Item 1 </li>
    <li> Item 2 </li>
    <li> Item 3 & lt;/li>
</ul>

Output

  • Item 1
  • Item 2
  • Item 3

Description List

A description list is defined with the <dl> tag. Each term is defined with the <dt> tag, and each description is defined with the <dd> tag.

<dl>
    <dt> HTML </dt>
    <dd> Hypertext Markup Language </dd>
    <dt> CSS </dt>
    <dd> Cascading Style Sheets </dd>
</dl>

Output

HTML
Hypertext Markup Language
CSS
Cascading Style Sheets

Nesting Lists

You can also create nested lists, which are lists within lists. To create a nested list, simply add another <ul> or <ol> tag inside an existing list item ( <li> ).

<ul>
    <li> Item 1 </li>
    <li> Item 2
        <ul>
             <li> Subitem 1 </li>
             <li> Subitem 2 </li>
         </ul>
     </li>
</ul>

Output

  • Item 1
  • Item 2
    • Subitem 1
    • Subitem 2

List Item Attributes

The <li> tag supports the global attributes. However, it does not support the type and value attributes (which are supported by the <ol> tag).

Example

<ol>
    <li value="3"> Item 1 </li>
    <li> Item 2 </li>
    <li> Item 3 </li>
</ol>

Output

  1. Item 1
  2. Item 2
  3. Item 3
