HTML Block and Inline Elements

Every HTML element has a default display value, depending on what type of element it is. The two most common display values are block and inline.

Block-level Elements

A block-level element always starts on a new line, and the browsers automatically add some space (a margin) before and after the element.

A block-level element always takes up the full width available (stretches out to the left and right as far as it can).

Two commonly used block elements are: <p> and <div>.

The <p> element defines a paragraph in an HTML document.

The <div> element defines a division or a section in an HTML document.

<div>
    <p> This is a paragraph. </p>
    <p> This is another paragraph. </p>
</div>

Output

This is a paragraph.

This is another paragraph.

Here are the block-level elements in HTML:

<div> , <p> <h1> , <h2> , <h3> , <h4> , <h5> , <h6> <header> , <footer> , <section> , <article> <main> , <aside> , <figure> , <figcaption> <table> , <tr> , <th> , <td> <ul> , <ol> , <li> <blockquote> , <address>

Inline Elements

An inline element does not start on a new line and only takes up as much width as necessary.

Two commonly used inline elements are: <a> and <span>.

The <a> element defines a hyperlink, which is used to link from one page to another.

The <span> element is a container for inline content, which does not inherently represent anything. It can be used to group elements for styling purposes (using the class or id attributes), or because they share attribute values, such as lang.

<h1> This is a heading </h1>
<p> This is a <a href="#"> link </a> inside a paragraph. </p>

Output

This is a heading

This is a link inside a paragraph.

Here are the inline elements in HTML:

<a> , <span> , <img> , <strong> , <em> , <b> , <i> , <u> , <small> , <sub> , <sup> , <code> , <samp> , <kbd> , <var> , <br> , <q> , <cite> , <abbr> , <time>
