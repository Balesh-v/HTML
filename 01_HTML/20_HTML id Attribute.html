HTML id Attribute

The HTML id attribute is used to specify a unique id for an HTML element. You cannot have more than one element with the same id in an HTML document.

The id attribute is used to uniquely identify an HTML element. It can be used by CSS and JavaScript to select and manipulate the element.

Example

<h1 id="main-heading"> This is the main heading </h1>
<p id="first-paragraph"> This is the first paragraph. </p>

Output

This is the main heading

This is the first paragraph.

The syntax for id is: write a hash character (#), followed by an id name. Then, define the CSS properties within curly braces {}.

Example

#main-heading {
  color: blue;
}
#first-paragraph {
  font-size: 20px;
}

Output

This is the main heading

This is the first paragraph.

Difference Between Class and ID

Class ID
A class can be used on multiple elements. An id must be unique and can only be used on one element.
A class is defined with a period (.) in CSS. An id is defined with a hash (#) in CSS.
A class is used to group elements. An id is used to identify a single element.
