HTML Iframes

An HTML iframe is used to display a web page within a web page.

HTML Iframe Syntax

The HTML <iframe> tag specifies an inline frame. An inline frame is used to embed another document within the current HTML document.

<iframe src="URL" title="description">

Output

Iframe - Set Height and Width

You can set the height and width of an iframe using the width and height attributes.

The height and width are specified in pixels by default:

<iframe src="01_Introduction.html" title="iframe example" width="600" height="400" style="border:1px solid black">
