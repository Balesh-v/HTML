Head Element

The <head> element is a container for metadata (data about data) and is placed between the <html> tag and the <body> tag.

Common Elements Inside the Head

  • <title>: Sets the title of the webpage, which is displayed in the browser's title bar or tab.
  • <meta>: Provides metadata such as character set, author, description, and keywords for search engines.
  • <link>: Links to external resources like stylesheets (CSS files).
  • <style>: Contains internal CSS styles for the webpage.
  • <script>: Links to or contains JavaScript code that adds interactivity to the webpage.

Example of a Head Element

<head> <meta charset="UTF-8"> <meta name="viewport" content="width=device-width, initial-scale=1.0"> <title>Head Element</title> <link rel="stylesheet" href="style.css"> </head>

Note

The content inside the <head> element is not displayed on the webpage itself, but it is essential for the proper functioning and SEO of the site.

The HTML <meta> Element

The <meta> element is used to provide metadata about the HTML document. Metadata is data (information) about data. The <meta> element is placed inside the <head> section of the document.

Common Uses of the Element

  • <meta charset="UTF-8"> : Specifies the character encoding for the HTML document.
  • <meta name="viewport" content="width=device-width, initial-scale=1"> : Ensures proper rendering and touch zooming on mobile devices.
  • <meta name="description" content="A brief description of the page"> : Provides a summary of the page content for search engines.
  • <meta name="keywords" content="HTML, CSS, JavaScript"> : Lists keywords relevant to the page content for search engines.
  • <meta name="author" content="Your Name"> : Specifies the author of the document.
