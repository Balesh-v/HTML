HTML Computer Code Elements

This section covers the various elements used in HTML to represent computer code.

<code> Element

The <code> element is used to define a piece of computer code. It is typically displayed in a monospaced font.

Example:

<code> console.log('Hello, World!'); </code>

Output

console.log('Hello, World!');

<pre> Element

The <pre> element is used to display preformatted text. Text within this element is displayed in a fixed-width font, and whitespace (such as spaces and line breaks) is preserved.

Example:

<pre> <code> x = 5; y = 6; z = x + y; </code> </pre>

Output

            
                 x = 5;
                 y = 6;
                 z = x + y;

<samp> Element

The <samp> element is used to define sample output from a computer program or system.

Example:

<samp> File not found: /path/to/file.txt </samp>

Output

File not found: /path/to/file.txt

<kbd> Element

The <kbd> element is used to define keyboard input. It is typically displayed in a monospaced font to indicate that it represents text entered by the user.

Example:

<kbd> Ctrl + C </kbd>

Output

Ctrl + C

<var> Element

The <var> element is used to define a variable in a mathematical expression or programming context. It is typically displayed in italics.

Example:

<var> x + y = z </var>

Output

x + y = z
