HTML Input Attributes

This chapter describes the different attributes for the HTML <input> element.

The value Attribute

The input value attribute specifies an initial value for an input field:

<input type="text" value="Hello World">

This will create a text input field with "Hello World" as the initial value.

The readonly Attribute

The input readonly attribute makes an input field non-editable:

<input type="text" value="Hello World" readonly>

This will create a text input field with "Hello World" as the initial value, but the user will not be able to change it.

The disabled Attribute

The input disabled attribute disables an input field, preventing user interaction:

<input type="text" value="Hello World" disabled>

This will create a text input field with "Hello World" as the initial value, but the user will not be able to interact with it.

The size Attribute

The input size attribute specifies the width of an input field in characters:

<input type="text" size="30">

This will create a text input field that is 30 characters wide.

The maxlength Attribute

The input maxlength attribute specifies the maximum number of characters allowed in an input field:

<input type="text" maxlength="10">

This will create a text input field that allows a maximum of 10 characters.

The min and max Attributes

The input min and max attributes specify the minimum and maximum values for numeric input fields:

<input type="number" min="1" max="10">

This will create a number input field that only accepts values between 1 and 10.

The placeholder Attribute

The input placeholder attribute provides a hint to the user about what to enter in the input field:

<input type="text" placeholder="Enter your name">

This will create a text input field with the placeholder text "Enter your name".

The multiple Attribute

The input multiple attribute allows users to select multiple files in a file input field:

<input type="file" multiple>

This will create a file input field that allows multiple file selections.

The pattern Attribute

The input pattern attribute specifies a regular expression that the input field's value must match for validation:

<input type="text" pattern="[A-Za-z]{3,}">

This will create a text input field that only accepts alphabetic characters with a minimum length of 3.

The required Attribute

The input required attribute specifies that an input field must be filled out before submitting the form:

<input type="text" required>

This will create a text input field that must be filled out before the form can be submitted.

The autofocus Attribute

The input autofocus attribute specifies that an input field should automatically get focus when the page loads.

<input type="text" autofocus>
