HTML Canvas

The HTML <canvas> element is used to draw graphics on a web page. The graphic to the left is created with <canvas>. It shows four elements: a red rectangle, a gradient rectangle, a multicolor rectangle, and a multicolor text.

What is HTML Canvas?

The HTML <canvas> element is used to draw graphics, on the fly, via scripting (usually JavaScript). The <canvas> element is only a container for graphics. You must use a script to actually draw the graphics. Canvas has several methods for drawing paths, boxes, circles, text, and adding images. Canvas is supported by all major browsers.

HTML Canvas Example

The following example creates a canvas element, and draws a red rectangle, a gradient rectangle, a multicolor rectangle, and multicolor text on it:

<canvas id="myCanvas" width="400" height="180" style="border:1px solid #000000;"></canvas>
